# faunaPetcare Clinic — Premium Pet Healthcare

A modern, premium veterinary care website built with React, TypeScript, and Tailwind CSS.

## Features

- Premium pet healthcare services showcase
- Online appointment booking
- Doctor profiles and testimonials
- Responsive design with dark mode support
- Smooth animations with Framer Motion

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
